How the Oil Markets Index Works

The Oil Markets Index (OMI) is a composite tracking global crude and refined-product benchmarks, rolled up from publicly available spot and futures prices.

Composite Math

OMI is computed as a weighted geometric mean of normalized price series: each input is indexed to 100 at its base date, then combined using fixed weights that reflect global consumption shares. Geometric mean is used so that a 10% rise in one benchmark and a 10% fall in another cancel cleanly, avoiding the arithmetic-mean drift bias.

Forward-Fill Only

When a benchmark is missing a print (weekend, holiday, source-side gap), we carry the last known value forward. We never interpolate or smooth — the line holds flat until the next real print arrives.

Overlay Comparisons

SPX, Gold, USD, and DXY overlays let you view crude relative to other macro anchors. Overlays are independently normalized and toggle on/off without recomputing OMI itself.

Time Machine Ranges

1D / 1W / 1M / 3M / 1Y / 5Y / MAX. All ranges use the same source data; longer ranges sample at daily granularity, shorter ranges at hourly where available.
